Coveo Ranking Information

The purpose of the Coveo Ranking Information is to give developers/clients/partners the ability to quickly check and validate the Ranking Information of search results.

How it works

Click on the extension so that it turns red to enable the ranking information.

The extension will try to add debug=true to the queries so that the Ranking Information is retrieved.
When an AJAX call is being made, it will also add context['fromCRI']=true; to the query.

In your Query Pipeline you then can set a condition to only enable debug=true when the Condition Context with key fromCRI is populated.

Do NOT set `debug=true` for all of your queries!!!

Usage

Goto the search page of your choice.

Now enable the Extension by clicking on it (the RED circle will indicate that the extension is active).

Execute a new query.
You now will see the Ranking information displayed.
The extension will try to insert the Ranking information into the existing results.

It will try to find the hyperlink or the title to match the Ranking Information with the results.
If it does not work properly: let us know.

You will see copy images on the table with the Ranking Information.
You can click on them to copy the field or the expression to the clipboard.

Now goto your Coveo Cloud Administration, goto the Query Pipelines, Result Ranking. Here you can copy the contents in: Filter by Rule or Query
